Usage Examples

Basic Embed

<iframe 
  src="https://your-chart-app.vercel.app/embed/chart-id" 
  width="600" 
  height="400"
  frameborder="0">
</iframe>

Responsive Embed

<div style="position: relative; width: 100%; height: 0; padding-bottom: 56.25%;">
  <iframe 
    src="https://your-chart-app.vercel.app/embed/chart-id" 
    style="position: absolute; top: 0; left: 0; width: 100%; height: 100%;"
    frameborder="0">
  </iframe>
</div>

Theme Support

<!-- Light theme -->
<iframe src="https://your-chart-app.vercel.app/embed/chart-id?theme=light"></iframe>

<!-- Dark theme -->
<iframe src="https://your-chart-app.vercel.app/embed/chart-id?theme=dark"></iframe>

<!-- System theme -->
<iframe src="https://your-chart-app.vercel.app/embed/chart-id?theme=system"></iframe>
