Basic Responsive Container:

<div style="width: 100%; max-width: 600px; margin: 0 auto; position: relative;">
  <div style="width: 100%; height: 0; padding-bottom: 66.67%; position: relative;">
    <iframe 
      src="https://chart-builder-one.vercel.app/embed/YOUR-CHART-ID" 
      style="position: absolute; top: 0; left: 0; width: 100%; height: 100%; border: 0;"
      title="Chart"
    ></iframe>
  </div>
</div>

With Styling:

<div style="width: 100%; max-width: 600px; margin: 0 auto; position: relative; border-radius: 8px; overflow: hidden; box-shadow: 0 2px 8px rgba(0,0,0,0.1); background: #ffffff;">
  <div style="width: 100%; height: 0; padding-bottom: 66.67%; position: relative;">
    <iframe 
      src="https://chart-builder-one.vercel.app/embed/YOUR-CHART-ID" 
      style="position: absolute; top: 0; left: 0; width: 100%; height: 100%; border: 0; border-radius: 8px;"
      title="Chart"
      loading="lazy"
    ></iframe>
  </div>
</div>

Modern Aspect Ratio:

<div style="width: 100%; max-width: 600px; aspect-ratio: 1.5; margin: 0 auto; border-radius: 8px; overflow: hidden; box-shadow: 0 2px 8px rgba(0,0,0,0.1);">
  <iframe 
    src="https://chart-builder-one.vercel.app/embed/YOUR-CHART-ID"

  ></iframe>
</div>

ðŸ’¡ Tips for Responsive Embeds

  • Aspect Ratio: Use padding-bottom percentage to maintain aspect ratio (height/width * 100)
  • Max Width: Set max-width to prevent charts from becoming too large on desktop
  • Mobile: Consider using calc(100% - 20px) for mobile spacing
  • Loading: Add loading="lazy" for better performance
  • Accessibility: Always include a descriptive title attribute
  • Modern Browsers: Use aspect-ratio CSS property when targeting newer browsers
