Responsive Chart Embed Examples

These examples demonstrate how embedded charts automatically adapt to different container sizes while maintaining their configured aspect ratio.

1. Full Width Container

Chart fills the entire width of its container, height adjusts automatically based on the chart's configured dimensions.

<div style="width: 100%; height: 400px;"> <iframe src="/embed/your-chart-id?theme=light" style="width: 100%; height: 100%; border: none;"> </iframe> </div>

2. Grid Layout

Charts in a responsive grid layout. Each chart maintains its aspect ratio while fitting the available space.

.grid { display: grid; grid-template-columns: 1fr 1fr; gap: 20px; } <div class="grid"> <iframe src="/embed/chart-1?theme=light"></iframe> <iframe src="/embed/chart-2?theme=dark"></iframe> </div>

3. Card Layout

Charts embedded in card-like containers with different sizes.

<!-- Small chart --> <div style="height: 200px;"> <iframe src="/embed/your-chart-id"></iframe> </div> <!-- Medium chart --> <div style="height: 300px;"> <iframe src="/embed/your-chart-id"></iframe> </div>

4. Theme Options

Charts can be embedded with different theme options: light, dark, or system (auto-detect).

Light Theme

Dark Theme

System Theme

<!-- Light theme --> <iframe src="/embed/your-chart-id?theme=light"></iframe> <!-- Dark theme --> <iframe src="/embed/your-chart-id?theme=dark"></iframe> <!-- System theme (auto-detect) --> <iframe src="/embed/your-chart-id?theme=system"></iframe> <!-- Default (no theme parameter) --> <iframe src="/embed/your-chart-id"></iframe>

5. Best Practices

  • Container Sizing: Always set a specific height or use CSS aspect-ratio for predictable sizing
  • Minimum Height: Charts have a minimum height of 300px for optimal readability
  • Responsive Design: Charts automatically adapt to container width while maintaining aspect ratio
  • Theme Integration: Use theme parameters to match your site's design
  • Loading States: Charts show loading spinners and error states gracefully
/* Recommended CSS for responsive charts */ .chart-embed { width: 100%; height: 400px; min-height: 300px; border: none; border-radius: 8px; box-shadow: 0 2px 8px rgba(0,0,0,0.1); } @media (max-width: 768px) { .chart-embed { height: 300px; } }
