📊 Responsive Chart Container - Production Ready

🎯 For Developers: This page contains production-ready CSS classes for responsive chart embedding. Copy the CSS and HTML patterns below for immediate use.

1. Standard Responsive Container (600:400 ratio)

<div class="responsive-chart-container"> <iframe src="https://your-chart-url" title="Chart" loading="lazy"></iframe> </div>

2. Wide Format Container (16:9 ratio)

<div class="responsive-chart-container-wide"> <iframe src="https://your-chart-url" title="Chart" loading="lazy"></iframe> </div>

3. Square Container (1:1 ratio)

<div class="responsive-chart-container-square"> <iframe src="https://your-chart-url" title="Chart" loading="lazy"></iframe> </div>

4. Minimal Container (no styling)

<div class="responsive-chart-container-minimal"> <iframe src="https://your-chart-url" title="Chart" loading="lazy"></iframe> </div>

5. Modern CSS Aspect Ratio (Chrome 88+, Firefox 89+, Safari 15+)

<div class="responsive-chart-container-modern"> <iframe src="https://your-chart-url" title="Chart" loading="lazy"></iframe> </div>

📋 Implementation Guide

Quick Setup (3 steps):

  1. Copy the CSS from the <style> section above
  2. Use the HTML pattern with your chart URL
  3. Remove fixed dimensions from iframes (no width/height attributes)
⚠️ Critical: Never use width or height attributes on iframes. They override the responsive CSS and cause mobile cutting issues.

Aspect Ratio Customization:

To create custom aspect ratios, calculate: padding-bottom: (height ÷ width × 100)%

  • 600×400: 66.67% (standard)
  • 800×600: 75% (4:3 ratio)
  • 1920×1080: 56.25% (16:9 ratio)
  • 500×500: 100% (1:1 square)
/* Custom aspect ratio example */ .my-custom-container::before { padding-bottom: 75%; /* 4:3 ratio */ }

Browser Support:

  • Standard method (::before): All modern browsers + IE11
  • aspect-ratio property: Chrome 88+, Firefox 89+, Safari 15+

🔧 Troubleshooting

Chart cut off on mobile?

  • Remove width and height attributes from iframe
  • Ensure proper viewport meta tag is present
  • Check for fixed container widths exceeding viewport

Charts not scaling?

  • Verify iframe has no fixed dimensions
  • Check CSS specificity isn't being overridden
  • Test with the minimal container first

Horizontal scrolling on mobile?

  • Use calc(100vw - 20px) instead of percentage widths
  • Add overflow-x: hidden to body
  • Ensure box-sizing: border-box is set
